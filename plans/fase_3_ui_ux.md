# Fase 3: Interfaz de Usuario y Experiencia (UI/UX)

## Objetivo
Crear una interfaz premium, moderna y responsiva utilizando el sistema de diseño Tailwind y efectos de vidrio.

## Sprint 1: Layout y Componentes Estructurales
- [ ] Diseñar Header flotante con el Logo (Minimalista Púrpura)
- [ ] Crear contenedor del mapa (Full screen)
- [ ] Diseñar tarjeta de detalles del ave (Modal o Panel Lateral)
    - [ ] Estado inicial: Oculto
    - [ ] Estado activo: Visible sobre el mapa

## Sprint 2: Estética y "Wow" Factor
- [ ] Implementar Glassmorphism en paneles:
    - Fondo semi-transparente (white/10 o purple/10)
    - `backdrop-filter: blur(10px)`
    - Bordes sutiles
- [ ] Estilizar marcadores del mapa (Iconos personalizados o colores del tema)
- [ ] Añadir micro-interacciones:
    - Hover effects en marcadores
    - Transiciones suaves (fade-in/slide-up) al abrir detalles
- [ ] Verificar contraste y legibilidad (Light Mode strict)
