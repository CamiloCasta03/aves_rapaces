# Fase 1: Fundamentos y Configuración

## Objetivo
Establecer la base tecnológica del proyecto, instalar dependencias y configurar el sistema de diseño.

## Sprint 1: Inicialización del Entorno
- [ ] Inicializar proyecto con Vite (Vanilla JS)
- [ ] Estructurar directorios (`/src`, `/public`, `/assets`)
- [ ] Instalar dependencias de producción:
    - `leaflet` (Mapas)
- [ ] Instalar dependencias de desarrollo:
    - `tailwindcss`
    - `postcss`
    - `autoprefixer`

## Sprint 2: Configuración del Sistema de Diseño
- [ ] Inicializar TailwindCSS (`npx tailwindcss init -p`)
- [ ] Configurar `tailwind.config.js`:
    - [ ] Definir paleta de colores (Púrpura primario, complementarios)
    - [ ] Configurar fuentes (Inter/Sans-serif)
    - [ ] Habilitar soporte para "Glassmorphism" (backdrop-blur utilities)
- [ ] Crear `style.css` principal con directivas de Tailwind
- [ ] Implementar reset CSS básico y fuentes globales
