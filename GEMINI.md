# Project Context: Aves Rapaces Map

## Overview
A web application to visualize Raptor Birds in Colombia using an interactive map.
User requests to focus on high-fidelity UI, "Premium" aesthetics, and a specific "Purple/Light" theme.

## Tech Stack
- **Frontend**: Vite + Vanilla JavaScript
- **Styling**: TailwindCSS (v3) + Custom CSS for Leaflet/Glassmorphism
- **Map**: Leaflet.js
- **Data**: Static JSON (`data/birds.json`)

## Key Features
- **Geographic Focus**: Colombia (Departments).
- **UI/UX**: Glassmorphism, smooth animations, fully responsive.
- **Content**: Bird details (Family, Image, Food).
